icDashBoard

Java Servlet/JSP dashboard-plattform för InfoCaption-supportorganisationer. Modulbaserad, databasdriven, med SSO, gruppstyrning och centraliserad konfiguration.

40 Servlets
17 Moduler
7 Filters
22 Utilities
43 SQL-migrationer
Java 21 + Tomcat 9

Security Layers

flowchart TB
    Req["Incoming Request"] --> RL["RateLimitFilter\n(brute-force protection)"]
    RL --> BS["BodySizeLimitFilter\n(max payload)"]
    BS --> CT["ContentTypeValidation\n(whitelist MIME types)"]
    CT --> ENC["EncodingFilter\n(UTF-8 enforce)"]
    ENC --> SH["SecurityHeaderFilter\n(CSP, HSTS, X-Frame, nonce)"]
    SH --> CSRF["CsrfFilter\n(token per session)"]
    CSRF --> Auth["AuthFilter\n(session check)"]

    Auth --> PW["BCrypt\n(work factor 12)"]
    Auth --> SSO["SAML2 SSO\n(Microsoft Entra ID)"]

    Auth --> Secrets["SecretsConfig\n(app-secrets.properties)"]
    Auth --> Crypto["CryptoUtil\n(AES-256-GCM)"]
